# StudyOnlinePlatForm
本项目是一个中大型网络教学平台架构及部分实现(教学互动，课程点播，教学直播)    

***在线预览地址***
## 目前实现功能
- 用户登录注册及鉴权
- 文件上传与下载
- 文档在线预览
- 教师在线备课编辑
- 课程视频点播
- 在线学习课程
- 后台用户管理
